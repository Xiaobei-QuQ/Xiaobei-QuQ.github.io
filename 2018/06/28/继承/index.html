继承

发表于 2018-06-28 • 分类于 JavaScript

面向对象有三大特性：继承，封装和多态。虽然js没有继承，但是可以利用原型链实现复用代码。

ES6的时候JS利用Class实现了继承。但是ES5的原型对前端来说更容易理解其原理。

继承

  1. 得到一个类的属性

属性获取

第一步，我们想实现上述Male类继承Person类的属性。对象属性的获取是通过构造函数的执行，在一个类中执行另外一个类的构造函数，就可以把属性赋值到自己内部，但是需要把环境改到自己的作用域内，这就要借助函数call了

方法获取

第二部，要想实现Male类继承Peron类的方法printName。类的方法都定义在了prototype里面，所以只要把子类的prototype改为父类的prototype的备份即可。

不能使用直接赋值，因为引用关系，这样会导致后续修改子类的prototype也修改了父类的prototype，修改的是一个值。

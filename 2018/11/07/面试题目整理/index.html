面试题目整理

发表于 2018-11-07 • 分类于 JavaScript

步步高面试

第一次技术面，仔细想想，很多问题只是略过，其实可以展开很多知识点。

  1. CSS盒模型
    • content-box: width == 内容区宽度
    • border-box: width == 内容区宽度 + padding 宽度 + border 宽度
  2. flex布局
  3. js作用域链
  4. 闭包，垃圾回收
  5. 变量声明提前，词法树
  6. DOM事件模型
  7. 事件绑定
  8. Vue生命周期
  9. Vue组件通信
  10. Vuex
  11. VueRouter
  12. 双向绑定

意识到自己的表现还是不太好，加多点自信和熟练吧

CVTE面试

一场良好却带遗憾的面试体验

  1. 闭包

  2. 垃圾回收（标记清除和引用清除）

  3. 跨域和同源策略（非简单请求和简单请求）

  4. 网站性能优化

  5. 事件循环event loop

    1. 问了主线程什么时候才会去执行任务队列的任务（回答执行完全部任务再去查找任务队列，有缺漏，应该是micro和macro任务两种情况）

  6. 清除浮动

  7. BFC

  8. 安全防范（cors，xss，https）

  9. ES6新特性（const和let）

  10. canvas

  11. HTTP缓存

  12. Vue一小部分知识点

  13. 编程题，要求传入数组[fn1,fn2,fn3]和callback，等fn1，fn2，fn3执行完毕再执行callback

  1. 编程题，EventEmit，写出构造函数把添加test和清除test和触发test

  2. 编程题，要求把父节点转换成子节点

  3. 遇到的最难的问题是什么，怎么解决。

总结： 第一面考得很多，很杂，但总体还好。面试官评价自己的知识深度不够。第二面没什么准备，基本节奏一直在面试官手里，有点没想到，到后面基本没在面，只是随便聊一聊。

总体说来第二面不是很满意，不能因为说第一面太累然后疏忽准备，应该把握节奏谈一下技术框架的知识。不能傻傻的等他问。

npm使用

–save和–save-dev命令，可以自动在package.json文件添加dependencies部分和devdependencies部分的模块和版本号

  • npm init会生成一个package.json文件，配置了需要发布的包的信息。
  • npm install –save 的包会成为该包的依赖写进package.json的dependencies部分
  • 在项目中利用.gitignore禁止node_modules上传github
  • 其他用户在使用时可以通过package.json文件npm install来安装所需模块

学习CSS的一些笔记

单列布局

  1. max-width: 1000px: 低于1000px时页面不会出现滚动条，且会随着width缩小，自己缩小
  2. width: 1000px; 低于1000px时页面会出现滚动条
  3. min-width: 1000px， 低于1000px，屏幕会出现滚动条，固定1000px，超过10000px占满整个屏幕. http://js.jirengu.com/lajuyerula/1/edit

圣杯布局

圣杯布局：

  1. 设置main的width为100%，aside和extra的width为定宽
  2. 三个块全部左浮动
  3. 左边asidemargin为-100%（相对父元素）；右边extramargin为-width（自己的width）=> 让aside和extra跑到main左右
  4. 给包裹三个块的content设置padding-left: -width(aside);margin-right: -width(extra); => 让main左右压缩
  5. aside相对自己left: -width; extra相对自己left: width; => 让aside和extra跑出来
  6. 清除浮动clearfix

优点:

  1. 是三列布局，两边固定宽度，中间自适应
  2. 中间内容元素在 dom 元素次序中优先位置

缺点:

  1. mian的最小宽度不能小于aside的宽度(因为设置父元素的padding时，aside和extra会被挤到下面去)

双飞翼布局:

双飞翼布局：

  1. 设置main的width为100%，aside和extra的width为定宽
  2. aside和extra和wrap三个块全部左浮动
  3. 左边asidemargin为-100%（相对父元素main）；右边extramargin为-width（自己的width）=> 让aside和extra跑到父元素main左边和右边；
  4. 给中间块的wrap设置margin-left:aside的width； margin-right: extra的width；
  5. 清除浮动clearfix

和圣杯布局的区别：
前三部基本相同，只是双飞翼中间块还加上了子元素wrap，设定wrap的margin来实现main的居中，而圣杯利用父元素的paddinhg。

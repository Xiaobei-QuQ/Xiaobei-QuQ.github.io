一. 媒体查询

  1. 注意顺序问题，width大的放上面，width小的放越下面，这样才可以覆盖前面的样式。

  2. 或者设置@media (min-width: ) and (max-width: )让彼此没有交集
    link: media 当媒体查询条件满足的时候可以引入css生效，注意link是提前下载好了css的，只是生效条件被限制。

  3. 通过media引入css文件

  4. http://js.jirengu.com/cihewozowu/1/

  5. 学会隐藏元素
    二. 响应式其实不实用，可以利用服务器返回不同网址来实现手机页面。

flex布局

  1. flex之前
    主要使用
    • normal flow
    • float+clear
    • position relative+absolute
    • display inline-block；
    • 负margin
  2. flex来了
    -flex与方向无关
    • 空间自动分配 元素自动对齐
    • 适用于简单的线形布局，更复杂的布局要交给grid布局
  3. flex基本概念
    轴 尺寸 起点 终点 flex item flex container
  4. flex container的六个属性
    • flex-direction 方向 row从左往右，挤在一行 row-reverse从右往左 column一个占一列，从上到下，column-reverse一个占 一列，从下到上。
    • flex-wrap 换行 默认是nowrap，wrap可以换行，父元素设置width，height，子元素都可以换行。
    • flex-flow 上面两个到简写
    • justify-content 主轴对齐方式 space-between把所有空间放在中间，space-around放周围，flex-start往起点靠，flex-end往终点靠， center，往中间靠
    • align-items 侧轴对齐方式 高度不能写死 stretch把所有元素伸展到和最高的一样高 flex-start 往起点靠 flex-end往终点靠，不延伸 center往中间靠
    • align-content 控制行与行之间的空隙 stretch默认，space-between，space-around，flex-start，flex-end，center。
  5. flex item的六个属性
    • flex-grow 分配多余的空间，按比例吃掉
    • flex-shrink 收缩比例
    • felx-basis 默认大小，占据宽度
    • flex的缩写
    • order 顺序，代替双飞翼布局
    • align-self 自身对齐

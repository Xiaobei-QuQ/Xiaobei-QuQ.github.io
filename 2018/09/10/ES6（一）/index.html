发表于 2018-09-10 • 分类于 JavaScript

复习ES6的一些知识

var-let-const

var会造成变量提升。

举个栗子，if里面使用a，else里面声明a，这样并不会报错

立即执行函数可以造出局部变量

ES6的let实现了局部变量

在块级作用域使用let的时候，没有使用let声明变量就使用就会报错，涉及到临时死区

let的作用域就是最近的{ }之间

不能用let声明两次变量

const只有一次赋值机会

函数里面使用作用域上的变量，要看函数执行的时机的变量的值

for循环和let赋值结合可以使用异步函数来使用不同的i，但是let在这里算是一个语法糖

在进入for循环的时候，for会把( )里面i的值赋值给{ }里面i的值，执行完就把{ }里面i的值返回给( )里面i的值

箭头函数

不加{ }时也不需要加return

箭头函数不传入this，使用的this仍然是外面的this

箭头函数抄袭coffee的箭头函数，方便使用函数的同时避免了this的混入

箭头函数对数组的增强是非常有表现力的

在vue选项属性回调时最好不要使用箭头函数的this

语法糖

函数默认参数 es5之前用||短路逻辑 es6用(a=0,b=0)传入默认参数

注意js传入默认参数比如一个空数组，每次调用都会产生一个空数组

剩余参数（将伪数组变为真数组）

  • Array.prototype.slice.call(arguments,2) //ES5
  • Array.from(arguments).slice(2)
  • function fn (a,b, …c){}

展开操作

  • […iterableObj] = [1, 3, 5, 7, 9];
  • [0, 2, …iterableObj, 4, 6, 8];

解构赋值

  • [a, b] = [b, a]
  • [a, b, …rest] = [10, 20, 30, 40, 50]
  • let {name, age} = frank
  • [a=5, b=7] = [1];
  • [a, b] = f()
  • [a, , b] = f();
  • var {p: foo, q: bar} = o
  • let {a = 10, b = 5} = {a: 3};
  • let {a:aa = 10, b:bb = 5} = {a: 3};
  • 对象浅拷贝 //尽量用… 不行就用object.assign()，最后才用for循环
  • 对象合并

对象属性加强

  • obj = { x, y }
  • obj = {[“baz” + quux() ]: 42}
  • 函数属性可以缩写

字符串

  3. 函数接字符串

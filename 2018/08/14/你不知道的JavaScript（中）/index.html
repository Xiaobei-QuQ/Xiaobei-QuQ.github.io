你不知道的JavaScript（中卷）

发表于 2018-08-14 • 分类于 JavaScript

很多JavaScript知识点，整理一下～

第一章 类型

七种类型： null undefined string number boolean object symbol

  • 变量的值没有类型，但它们持有的值有类型，类型定义了值的行为特征。

  • typeof undefined === ‘undefined’

  • typeof null === ‘true’

  • undefined !== undeclared 声明为赋值的为undefined，未声明报错为undeclared

  • 处理undeclared if ( DEBUG ) { } ==> if ( typeof DEBUG !== ‘undefined’){ }

通过typeof的安全防范机制（阻止报错）来检测undeclared变量，是个不错的选择。

第二章 值

将伪数组变成数组的方法

  • Array.prototype.slice.call(arguments)

  • Array.from(..)

字符串经常被当成字符数组，但是字符串不可变，数组可变，两者不一样

字符串反转例子，利用数组的reverse()成员函数

Js的数字类型是基于IEEE 754标准来实现的，双精度浮点数

  • toFixed 小数部分的显示位数

  • toPrecision 有效数位的显示位数

  • toExponential 指数格式显示

  • 42..toFixed(3)正确 42.toFixef(3) 错误

  • 0.1+0.2 在存储为二进制时是一个无理数，所以等于0.30000000000000004

  • undefined指从未赋值，null指曾赋过值，目前没有值

  • null是一个关键字，不是标识符，不能当做变量来使用和赋值

  • undefined是一个标识符，可以使用和赋值

  • 简单标量基本类型的值（字符串和数字）通过值复制来赋值/传递，而复合值（对象等）通过引用复制来赋值/传递。JavaScript中的引用不能指向别的变量/引用，只能指向值
  • 将数组a = [1,2,3]作为参数x传递给函数，函数里x可以push会改变数组a，但是x重新赋值为其他不影响a，因为只是将引用a的一个复本赋值给x，而a仍然指向[1,2,3]，我们不能通过引用x来更改引用a的指向，只能更改a和x共同指向的值
